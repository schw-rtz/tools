------------------------------------------------------------------------

dbTool - Database interaction

Commands:

    * about
          o Displays information on various topics
    * charset
          o shows character encodings
    * connection
          o Displays connection settings
    * export
          o Exports record, and dependant records to xml
    * help
          o Displays help on various commands.
    * query
          o Executes an sql query
